# Каталог товаров

SPA для поиска и просмотра товаров. React 18, TypeScript, FSD, SCSS.

## Запуск

```bash
npm install
npm run dev
```

Открой [http://localhost:5173](http://localhost:5173)

## Скрипты

- `npm run dev` — разработка
- `npm run build` — сборка
- `npm run test` — тесты
- `npm run lint` — проверка кода
